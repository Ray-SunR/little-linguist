# T003 Add global styles and theme tokens

**Phase**: Setup
**Depends on**: T001

## Task

Add base global styles and theme tokens for the reader UI.

## Acceptance Criteria

- `styles/globals.css` exists
- CSS variables or Tailwind tokens are defined for colors, radius, and shadows
- Default font size supports large, kid-friendly text

## Instructions

1. Create `styles/globals.css` if it does not exist.
2. Define a small set of theme tokens (color, radius, shadow).
3. Ensure readable base font sizing.
