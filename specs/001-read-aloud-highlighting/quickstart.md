# Quickstart: Core Reader MVP

## Prerequisites

- Node.js 20 LTS
- Package manager (npm or pnpm)

## Run the app (once code exists)

```bash
npm install
npm run dev
```

## Verify core flow

1. Open the reader page in a mobile-sized viewport.
2. Select a book from the static JSON list.
3. Press Read Story and confirm word highlighting advances.
4. Pause, resume, and stop to confirm state transitions.
5. Confirm highlight clears on stop and errors show a banner.
